# ℹ️ monorepo-pnp-jest-example

ℹ️ Example project with...

- Node.js
- Monorepo by Yarn workspaces
- Yarn PnP (Berry)
- Jest
- ESLint
- TypeScript

## Requires

- Node.js >= 14.19.0
- Yarn >= 2.4.3

## Install

```sh
npm i -g yarn@berry
yarn install
```

## Linting

```sh
yarn run lint
yarn run lint:fix # Lint and auto-fix
```

## Testing

```sh
yarn run test
```

Currently, the command works as an alias for the `yarn run lint` command.
